Available Tools

Web Search

search

Perform web searches using the Brave Search API. This tool provides comprehensive search results from across the web, with a focus on privacy and relevance.

Parameters:

  • query: Search query string
  • count: Number of results to return (optional)

Returns:

  • Search results with titles and descriptions
  • URLs for each result
  • Relevance scores
  • Additional metadata when available
